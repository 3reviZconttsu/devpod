## Quickstart with VSCode

Make sure you have installed [DevPod](./install.mdx) on your system.

### Install Prerequisites

Make sure to install the following prerequisites installed:
* [VSCode](https://code.visualstudio.com/download)
* [Remote ssh extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh)
* [Code CLI](https://code.visualstudio.com/docs/editor/command-line)

### Start a Workspace with VSCode

Navigate to 'Workspaces' > '+ Create'. Enter your project url or choose one of the quickstart examples. Make sure to select 'VSCode' under 'Advanced Options'
